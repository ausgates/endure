usage
=====

you'll need git to clone the repo
and ruby to play the game

```bash
git clone https://github.com/trommel/ravent
cd ravent/app/bin/lib/
ruby ravent.rb [ easy | medium | hard ]
```

or check ravent's version

```bash
ruby ravent.rb -v
```
